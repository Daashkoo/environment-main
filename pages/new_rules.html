List of New CSS Rules

Here are the 10 new CSS rules I applied to my website:

  • Fade-in Header (header): Added a fade-in animation for the header with the @keyframes fadeIn. This gives the header a smooth fade-in effect when the page loads.
  • Text Hover Scale in Header (header h1): When hovering over the header's title, it scales up slightly, providing a subtle interactive effect.
  • Logo Hover Effect (#logo): The logo will scale up when hovered, giving a subtle zoom-in effect to indicate interactivity.
  • Navigation Hover Glow (nav a): When hovering over navigation links, a glowing effect (text-shadow) is applied, and the link scales up slightly.
  • Main Hover Shadow and Lift (main): When hovering over the main content, it lifts slightly and has an enhanced shadow for a floating effect.
  • Paragraph Fade-in Effect (p): Paragraphs start with an opacity of 0 and fade in with the fadeInText animation after a short delay.
  • Footer Hover Effect (footer p): When the footer is hovered, the text in the footer slightly lifts to give a sense of interactivity.
  • Form Input Focus Effect (form input:focus, form textarea:focus): Inputs and text areas expand slightly (scale(1.02)) when focused, providing a clear visual cue to the user.
  • Submit Button Hover Effect (form input[type="submit"]:hover): When the submit button is hovered, it slightly scales up for a more interactive feel.
  • Main Hover Shake Effect (main): When the `main` content is hovered, it will shake slightly for a fun interactive effect.
  • Zoom-in Effect for h2: When hovering over `h2` elements, they will zoom in slightly to emphasize the heading.
